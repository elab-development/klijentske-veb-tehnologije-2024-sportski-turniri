# Sportski Turniri

Klijentska web aplikacija za pregled i organizaciju sportskih turnira.

## Tehnologije

- React
- TypeScript
- React Router DOM

## Funkcionalnosti

- Pregled turnira
- Search/filter
- Pagination
- Tournament details
- Responsive dizajn
- Routing
- Local storage login

## Pokretanje projekta

npm install

npm run dev
